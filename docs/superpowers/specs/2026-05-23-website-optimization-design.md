# 个人主页优化设计方案

## 概述

对现有纯静态个人主页进行全面优化升级，作为前端技术练习项目。保留原生 HTML + CSS + JS 技术栈，引入现代 CSS 特性和纯原生 JavaScript 交互效果，不依赖任何第三方库。

## 视觉风格

- **主题**：深色太空风格，蓝紫渐变为主色调
- **卡片效果**：毛玻璃质感（`backdrop-filter: blur`）
- **背景**：深色渐变 + Canvas 粒子动画
- **配色**：
  - 主背景：`#0a0a1a` → `#1a1a2e`
  - 主色：`#6366f1` → `#a855f7`（蓝紫渐变）
  - 次要色：`#06b6d4`（青色）
  - 文字：白/灰层次分明
- **支持暗色/亮色模式切换**，偏好存入 localStorage

## 技术方案

### HTML
- 语义化标签：`<nav>` `<section>` `<footer>` 等
- 无障碍基础：`aria-label` 等属性

### CSS
- 自定义属性管理主题色（`--primary`, `--bg`, `--text` 等）
- 毛玻璃效果：`backdrop-filter: blur(12px)`
- 滚动触发渐入动画（配合 JS Intersection Observer）
- 响应式设计：桌面 → 平板 → 手机三档适配
- 过渡动画：`transition` 统一管理

### JavaScript (纯原生)
- 时间问候语（保留现有逻辑并增强）
- 打字机效果：逐字输出副标题
- 粒子背景：Canvas 绘制浮动粒子
- 滚动渐入：Intersection Observer 驱动元素可见性动画
- 技能进度条：滚动到可视区域时从 0 填充到目标值
- 暗色/亮色切换：CSS 变量切换，localStorage 持久化
- 回到顶部按钮：滚动超过一屏时显示
